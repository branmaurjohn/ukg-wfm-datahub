# Report Catalog

This is the living index of reports your team publishes (internal or external).

## How to use this page
- If a report exists, it should be listed here.
- If it’s not listed here, it’s not “official.”
- Every report should link to its KPI contract(s).

---

## Catalog (Template)

| Report | Purpose | Grain | KPIs | Owner | Validation Status |
|---|---|---:|---|---|---|
| Overtime Utilization | OT trends by org | org-day | OT Hours | <owner> | Draft |
| Schedule vs Worked Variance | Plan vs reality | org-day | Variance Hours | <owner> | Draft |
| Accrual Health Monitor | balance integrity | person-day | Accrual Balance | <owner> | Draft |
| Approval Cycle Time | workflow bottlenecks | request-event | Cycle Time | <owner> | Draft |

---

## Validation Status rules
- **Draft**: defined but not fully reconciled
- **Validated**: sampled and reconciled
- **Certified**: validated + owned + monitored for drift
